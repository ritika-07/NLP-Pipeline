# NLP-Pipeline
### NLP pipeline implementation using SpaCy
### Steps:
1. Sentence segmentation
2. Tokenisation
3. Parts of Speech tagging
4. Lemmatisation
5. Stopword removal
6. Dependency parsing
7. Noun phrase recognition
8. Named Entity recognition
9. Coreference resolution (optional)
